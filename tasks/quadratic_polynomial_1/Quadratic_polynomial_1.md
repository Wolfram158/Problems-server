### Description
Let $`x_1`$, $`x_2`$ be the roots of polynomial $`x^2 + px + q`$. Calculate $`x_1^n + x_2^n`$ mod $`998244353`$ knowing that $`n`$ is non-negative integer less than $`2^{63}`$.
### Input format
First line: $`p`$, $`q`$, $`n`$ 
### Some facts
$`-2^{63}\le p,q \le 2^{63}-1`$
### Examples
|              Input               |  Output  |
|:--------------------------------:|:--------:|
|        $`3`$ $`4`$ $`0`$         |  $`2`$   |
|       $`17`$ $`-9`$ $`1`$        | $`-17`$  |
| $`-23432`$ $`-32531`$ $`144051`$ | $`5075`$ |
